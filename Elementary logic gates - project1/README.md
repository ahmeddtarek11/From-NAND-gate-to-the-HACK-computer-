<h1>project1 -  Elementary logic gates </h1>

This project involves designing and implementing basic computer logic gates using HDL (Hardware Description Language). The gates include:

- AND
- OR
- XOR
- NOT
- NAND
- NOR
- Multiplexers
- Demultiplexers
